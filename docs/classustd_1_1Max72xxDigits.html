Detailed Description

The MAX72XX Digits Display Class.

This class derived from Print class provides an implementation of a 7 segment digits display based on 8 digits modules driven by a maxim MAX7219 or MAX7221 controller connected over SPI.

  • See https://datasheets.maximintegrated.com/en/ds/MAX7219-MAX7221.pdf

void ustd::Max72xxDigits::write ( )
inline

Flushes the frame buffer to the display

In order to implement flicker free double buffering, no display function has any immediate effect on the display. All display operations are buffered into a frame buffer. By calling this method, the current content of the frame buffer is displayed.
